HTML

HTML (HyperText Markup Language):
- Purpose: Defines the structure and content of a webpage.
- Example: Text, images, links.

CSS

CSS (Cascading Style Sheets):
-Purpose: Styles the appearance of the webpage.
-Example: Colors, fonts, layouts.

JavaScript

JavaScript:
-Purpose: Adds interactivity and dynamic behavior to a webpage.
-Example: Animations, form validation, interactive elements.
